# Počasie

Zobrazí aktuálne počasie a predpoveď na nasledovné dni pre zadané mesto/GPS súradnice. Dáta pre počasie poskytuje služba [met.no](https://www.met.no/en).

## Nastavenia aplikácie

V tejto časti je možné nastaviť:

- Počet dní
- Miesto
- Zem. šírku
- Zem. dĺžku
